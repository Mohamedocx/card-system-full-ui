# 🪪 نظام موحد لإنشاء بطاقات الطلاب

نظام متطور وسهل الاستخدام لإنشاء بطاقات طلاب احترافية من ملف Excel وقوالب صور، مع دعم كامل للنصوص العربية والباركود.

---

## ✨ المميزات الرئيسية

✅ **إنشاء بطاقات احترافية** - من خلال دمج ملف Excel مع قالب صورة  
✅ **دعم كامل للعربية** - معالجة النصوص العربية والخطوط العربية المخصصة  
✅ **إضافة باركود** - توليد رموز QR/Barcode تلقائياً  
✅ **تخطيط B4 مرن** - تخصيص عدد الأعمدة والصفوف والمسافات  
✅ **إنتاج متعدد الصيغ** - صور PNG + ملفات PDF  
✅ **دعم البطاقات الثنائية** - واجهة أمامية وخلفية للبطاقة  
✅ **واجهة ويب سهلة** - تطبيق Streamlit بدون الحاجة لسطر الأوامر  
✅ **تحميل مباشر** - تنزيل الملفات الناتجة بسهولة

## 📦 التثبيت

### 1. استنساخ أو تحميل المشروع
```bash

```

### 2. إنشاء بيئة افتراضية (اختياري لكن موصى به)
```bash

```

### 3. تثبيت المكتبات المطلوبة
```bash

```

أو قم بتثبيتها يدويًا:

## 🚀 طريقة الاستخدام

### تشغيل التطبيق
```bash
streamlit run student_card_system.py
```

الخطواتتطبيق سيفتح في المتصفح تلقائياً على `http://localhost:8501`

### خطوات الاستخدام داخل التطبيق

#### 1️⃣ **إعداد الملفات المطلوبة**
- **ملف Excel**: قائمة بأسماء الطلاب والأرقام/المسلسلات
  - يجب أن يحتوي على عمودين على الأقل: الاسم والرقم
  - الصيغ المدعومة: `.xlsx` أو `.xls`

- **قالب الوجه الأمامي**: صورة تصميم البطاقة الأمامية
  - الصيغ المدعومة: PNG، JPG، JPEG
  - الحجم الموصى به: نفس حجم البطاقة المطلوبة (عادة ما يكون 85.6 x 53.98 ملم)

- **قالب الخلفية** (اختياري): صورة ظهر البطاقة
  - نفس متطلبات الوجه الأمامي

- **خط عربي** (اختياري): ملف الخط العربي المخصص
  - الصيغ المدعومة: TTF، OTF
  - الخطوط الموصى بها: NeoSansArabic أو Droid Sans Arabic

- **خط إنجليزي** (اختياري): ملف الخط الإنجليزي/اللاتيني
  - الصيغ المدعومة: TTF، OTF

#### 2️⃣ **ربط الأعمدة**
- اختر عمود الاسم من ملف Excel
- اختر عمود الرقم/السيريال
- فعّل معالجة العربية إذا لزم الأمر

#### 3️⃣ **تخصيص النصوص**
- حدد حجم خط الاسم والرقم
- اختر ألوان النصوص
- قرر ما إذا كنت تريد عرض الاسم والرقم

#### 4️⃣ **إضافة الباركود**
- فعّل إضافة الباركود إذا أردت
- اختر حجم وموضع الباركود
- اختر ما إذا كنت تريد عرض الرقم أسفل الباركود

#### 5️⃣ **تحديد المواقع**
- حدد مواقع النصوص والباركود على البطاقة (باستخدام الإحداثيات بالبكسل)
- استخدم معاينة تجريبية أو قياس يدوي

#### 6️⃣ **تخطيط صفحات الطباعة**
- عدد الأعمدة والصفوف لكل صفحة A4
- الهوامش والمسافات بين البطاقات

#### 7️⃣ **إنشاء البطاقات**
- اضغط زر "إنشاء البطاقات"
- انتظر اكتمال المعالجة

#### 8️⃣ **تحميل النتائج**
- تحميل صور PNG الفردية
- تحميل ملفات PDF المنسقة (أمامي فقط، أو أمامي + خلفي مدمج)
- تحميل جميع الملفات كملف ZIP واحد

## 🔧 الإعدادات المتقدمة

### متغيرات الوضوح (DPI)
- الدقة الافتراضية: **300 DPI** (جودة طباعة احترافية)
- جودة المخرجات مناسبة للطباعة التجارية

### ألوان النصوص
استخدم المنتقي الملونelect أي لون تريده، أو أدخل كود HEX مباشرة (#FFFFFF للأبيض، #000000 للأسود، إلخ)

### معالجة النصوص العربية
- تشكيل العربية يحل مشكلة الأحرف المعزولة والمتصلة
- يضمن عرض صحيح للنصوص العربية المعقدة

|------|-----------|-------|
| أحمد محمد | 001 | أول |
| فاطمة علي | 002 | أول |
| علي حسن | 003 | ثانى |

## 🎨 نصائح للحصول على أفضل النتائج

1. **حجم القالب**: تأكد من أن قالب الصورة بحجم مناسب (عالي الدقة موصى به 300 DPI)
2. **ألوان قابلة للقراءة**: استخدم ألوان متناقضة (أسود على أبيض أو أبيض على أسود)
3. **حجم الخط**: اختبر أحجام الخطوط على عينة قبل إنشاء جميع البطاقات
4. **الخطوط المدعومة**: إذا لم تظهر الخطوط، تأكد من رفع ملف الخط الصحيح
5. **الباركود**: تأكد من أن أرقام الطلاب فريدة (لتجنب رموز متطابقة)

## 🐛 استكشاف الأخطاء والمشاكل الشائعة

### المشكلة: "الأعمدة التالية غير موجودة في ملف الإكسل"
**الحل**: تأكد من اختيار أسماء الأعمدة الصحيحة من القائمة المنسدلة

### المشكلة: النصوص العربية تظهر بشكل غريب
**الحل**: فعّل "تفعيل تشكيل/معالجة العربية" أو رفع خط عربي مخصص

### المشكلة: الخط لا يظهر كما متوقع
**الحل**: رفع ملف الخط (TTF أو OTF) من جهازك

### المشكلة: الباركود منقوط أو غير واضح
**الحل**: زيادة قيمة "ارتفاع الباركود الداخلي" (module_height)

### المشكلة: البطاقات لا تتمحور بشكل صحيح على الصفحة
**الحل**: اضبط الهوامش والمسافات تحت "تخطيط صفحات A4"
